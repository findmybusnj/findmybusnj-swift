# SDevFlatColors
[SDevFlatColors Swift](https://github.com/0x73/SDevFlatColors)

## Usage
Usage is really simple.
````
UIView *testView = [[UIView alloc] initWithFrame:[UIScreen mainScreen].bounds];
testView.backgroundColor = [UIColor flat:Turquoise];
````
